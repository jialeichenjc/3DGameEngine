# Pong Game
Follows the same flow as the classic pong game
For 2 players

## Control
Right paddle: 
O Key: Up
K Key: Down

Left paddle:
W Key: Up
S Key: Down

Restart Game: Space
Resume Game: Space

## Win condition:
Who ever scores 5 points first wins
